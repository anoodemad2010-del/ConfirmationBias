# Perspective Shift — Confirmation Bias Educational Site

An IGCSE Global Perspectives project. A multi-page educational website about confirmation bias — what it is, why it matters, and how to think more clearly.

**Stack:** React 19 · TanStack Router · Vite · Tailwind CSS v4 · TypeScript · shadcn/ui

---

## Running Locally

**Requirements:** Node.js 18+ (or Bun)

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev
# → http://localhost:5173
```

### Other commands

```bash
npm run build      # Production build → /dist
npm run preview    # Preview production build locally
npm run lint       # Run ESLint
npm run format     # Run Prettier
```

## Deploying

### Cloudflare Pages (recommended)

```bash
npm run build
npx wrangler pages deploy dist --project-name confirmation-bias
```

Or connect your GitHub repo in the [Cloudflare Pages dashboard](https://pages.cloudflare.com/) with:
- Build command: `npm run build`
- Output directory: `dist`

### Netlify

Push to GitHub, connect in Netlify dashboard — `netlify.toml` is pre-configured.

Or via CLI:
```bash
npm run build
npx netlify deploy --prod --dir=dist
```

### Vercel

```bash
npm run build
npx vercel --prod
```

`vercel.json` is pre-configured for SPA routing.

### GitHub Pages

```bash
npm run build
# Install gh-pages: npm install -D gh-pages
npx gh-pages -d dist
```

> **Note:** All platforms need a rewrite rule `/* → /index.html` for client-side routing.
> `public/_redirects`, `netlify.toml`, and `vercel.json` handle this automatically.
